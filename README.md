# International Dating - Dating service for school/university events.

This is telegram bot which help find couple on "Winter ball". Main 
goal of this project is help to students find pair and make 
event planning more simple.

Main functions: 
- Dating service.
- CRM for event organisers.

Used stack/frameworks:
- C# .NET 9.0
- TelegramBot API 22.0
- Entity framework 9.0
- Google API(for integration data in google sheets)
